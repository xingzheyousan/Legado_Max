# Android 底部栏液态玻璃效果实现方案

基于 Legado 项目的底部导航栏玻璃质感实现。

---

### libs.versions.toml

## 2. 布局结构（三层叠加）

核心思路：`LiquidGlassView` + `半透明渐变 overlay` + `实际内容 View` 三层叠加。

```xml
<FrameLayout
    android:id="@+id/bottom_navigation_glass"
    android:layout_width="match_parent"
    android:layout_height="56dp"
    android:elevation="8dp">

    <!-- 第一层：玻璃折射效果 -->
    <com.qmdeve.liquidglass.widget.LiquidGlassView
        android:id="@+id/bottom_navigation_glass_view"
        android:layout_width="match_parent"
        android:layout_height="match_parent"
        android:clickable="false"
        android:focusable="false" />

    <!-- 第二层：半透明渐变 overlay（增加质感） -->
    <View
        android:id="@+id/bottom_navigation_shell_overlay"
        android:layout_width="match_parent"
        android:layout_height="match_parent"
        android:background="@drawable/bg_liquid_glass_shell_overlay"
        android:clickable="false"
        android:focusable="false" />

    <!-- 第三层：实际内容（底部导航栏） -->
    <BottomNavigationView
        android:id="@+id/bottom_navigation_view"
        android:layout_width="match_parent"
        android:layout_height="match_parent"
        android:background="@android:color/transparent"
        app:labelVisibilityMode="unlabeled" />

</FrameLayout>
```

## 4. Kotlin 配置代码

```kotlin
private fun setupLiquidGlass() {
    binding.run {
        // 绑定背景容器（用于折射采样，必须）
        bottomNavigationGlassView.bind(contentContainer)
        
        bottomNavigationGlassView.setCornerRadius(24f.dpToPx())
        bottomNavigationGlassView.setBlurRadius(12f.dpToPx())
        bottomNavigationGlassView.setRefractionHeight(72f.dpToPx())
        bottomNavigationGlassView.setRefractionOffset(46f.dpToPx())
        bottomNavigationGlassView.setDispersion(0.15f)
        bottomNavigationGlassView.setTintAlpha(0.25f)
        bottomNavigationGlassView.setDraggableEnabled(false)
        bottomNavigationGlassView.setElasticEnabled(true)
        bottomNavigationGlassView.setTouchEffectEnabled(true)
        bottomNavigationGlassView.isClickable = false
        bottomNavigationGlassView.isFocusable = false
        bottomNavigationGlassView.invalidate()
    }
}
```

---

## 5. 关键参数说明

| 参数 | 作用 | 调低效果 | 调高效果 |
|------|------|----------|----------|
| `refractionHeight` | 折射高度 | 玃璃感减弱 | 玻璃感增强 |
| `blurRadius` | 模糊强度 | 背景更清晰 | 背景更模糊 |
| `dispersion` | 色散程度 | 边缘彩虹减弱 | 边缘彩虹明显 |
| `tintAlpha` | 着色透明度 | 更透明 | 更有颜色倾向 |

### 只想要模糊+透明（不要玻璃质感）

```kotlin
glassView.setRefractionHeight(0f)
glassView.setRefractionOffset(0f)
glassView.setDispersion(0f)
glassView.setBlurRadius(25f)
glassView.setTintAlpha(0.3f)
```

## 6. 底栏管理界面逻辑

### 核心设计思路

- **默认使用原底栏**：软件启动时使用默认配置
- **只有用户修改才生效**：用户在底栏管理界面选择/编辑方案后才改变效果
- **方案化管理**：每个配置是一个独立的"方案包"，可切换、编辑、删除

### 数据结构

```kotlin
// 方案配置
data class Config(
    var name: String,              // 方案名称
    var isNightMode: Boolean,      // 是否夜间模式
    var layoutMode: String = "floating",  // 布局模式：floating/sidebar
    var effectMode: String = "glass",     // 效果模式：glass/frosted/solid
    var opacity: Int = 76,         // 透明度 0-100
    var icons: MutableMap<String, String> = linkedMapOf()  // 自定义图标
)

// 方案条目
data class Entry(
    val config: Config,
    val source: Source,            // 来源：BUILTIN/LOCAL
    val dirName: String            // 方案目录名，"default" 为默认方案
)
```

### 默认方案配置

```kotlin
private fun defaultEntry(isNight: Boolean): Entry {
    return Entry(
        Config(
            name = "默认",
            isNightMode = isNight,
            layoutMode = "floating",
            effectMode = "glass",   // 默认玻璃效果
            opacity = 76            // 默认透明度
        ),
        Source.BUILTIN,
        "default"                  // 默认方案标识
    )
}
```

### AppConfig 存储当前配置

```kotlin
object AppConfig {
    // 当前效果模式，默认 "glass"
    var bottomBarEffectMode: String
        get() = appCtx.getPrefString(PreferKey.bottomBarEffectMode, "glass")
            ?.takeIf { it in setOf("glass", "frosted", "solid") } ?: "glass"
        set(value) {
            appCtx.putPrefString(PreferKey.bottomBarEffectMode, value)
        }
    
    // 玻璃效果透明度，默认 68
    var liquidGlassLevel: Int
        get() = appCtx.getPrefInt(PreferKey.liquidGlassLevel, 68).coerceIn(0, 100)
        set(value) { appCtx.putPrefInt(PreferKey.liquidGlassLevel, value) }
    
    // 当前激活的方案名，默认 "default"
    fun activeDirName(isNight: Boolean): String {
        return appCtx.getPrefString(
            if (isNight) PreferKey.navigationBarPackageNight else PreferKey.navigationBarPackageDay,
            "default"
        )
    }
}
```

### 应用方案（切换效果）

```kotlin
fun apply(entry: Entry) {
    val config = entry.config
    
    // 记录当前激活的方案
    val key = if (config.isNightMode) activeNightKey else activeDayKey
    appCtx.putPrefString(key, entry.dirName)
    
    // 更新全局配置
    AppConfig.bottomBarEffectMode = config.effectMode
    AppConfig.liquidGlassLevel = config.opacity
    
    // 发送事件通知主界面刷新
    postEvent(EventBus.NAVIGATION_BAR_CHANGED, config.isNightMode)
}
```

### 管理界面核心功能

```kotlin
class NavigationBarManageActivity : BaseActivity() {
    
    // 加载方案列表
    private fun loadPackages() {
        val entries = loadEntries(isNightMode)
        adapter.submit(entries, activeDirName)
    }
    
    // 编辑方案对话框
    private fun showEditDialog(entry: Entry?) {
        // 可编辑：名称、布局模式、效果模式、透明度、图标
        // 效果模式选项：纯色(solid)、玻璃(glass)、磨砂(frosted)
        selector("效果模式", listOf("纯色", "玻璃", "磨砂")) { _, index ->
            config.effectMode = when (index) {
                0 -> "solid"
                2 -> "frosted"
                else -> "glass"
            }
        }
        
        // 透明度调节
        NumberPickerDialog()
            .setMinValue(0)
            .setMaxValue(100)
            .setValue(config.opacity)
            .show { config.opacity = it }
    }
    
    // 应用方案
    private fun applyPackage(entry: Entry) {
        NavigationBarIconConfig.apply(entry)
        postEvent(EventBus.NAVIGATION_BAR_CHANGED, entry.config.isNightMode)
        loadPackages()
    }
}
```

### 主界面响应配置变化

```kotlin
class MainActivity : BaseActivity() {
    
    override fun observeLiveBus() {
        observeEvent<Boolean>(EventBus.NAVIGATION_BAR_CHANGED) {
            setupLiquidGlass()  // 根据新配置重新设置效果
        }
    }
    
    private fun setupLiquidGlass() {
        val effectMode = AppConfig.bottomBarEffectMode
        val glassLevel = AppConfig.liquidGlassLevel
        
        when (effectMode) {
            "solid" -> {
                // 禁用玻璃效果，使用纯色背景
                bottomNavigationGlassView.visibility = View.GONE
                bottomNavigationShellOverlay.background = createSolidDrawable()
            }
            "frosted" -> {
                // 磨砂效果（参数不同）
                setupGlassView(refractionHeight = 0f, blurRadius = glassLevel * 0.5f)
            }
            "glass" -> {
                // 玻璃效果
                setupGlassView(refractionHeight = glassLevel * 1.06f)
            }
        }
    }
}
```

## 7. 效果模式说明

| 模式 | 效果 | 实现 |
|------|------|------|
| `glass` | 液态玻璃（折射+模糊） | LiquidGlassView 全参数启用 |
| `frosted` | 磨砂玻璃（仅模糊） | refractionHeight=0，保留 blur |
| `solid` | 纯色渐变 | LiquidGlassView 隐藏，使用普通 drawable |

参考以下方案实现底部栏液态玻璃效果及管理界面：

## 玻璃效果实现
1. 使用 com.qmdeve.liquidglass:core 库
2. 布局三层叠加：LiquidGlassView + overlay + BottomNavigationView
3. overlay 渐变：#46FFFFFF -> #2AFFFFFF -> #261E2A36
4. 配置参数：cornerRadius 24dp, blurRadius 12dp, refractionHeight 72dp

## 管理界面逻辑
1. 默认使用 "default" 方案
2. 方案数据结构：Config(name, effectMode, opacity, icons)

5. 用户在管理界面切换方案后，发送事件通知主界面刷新
6. 主界面根据 AppConfig.bottomBarEffectMode 决定显示哪种效果

## 关键点
- 软件启动时默认使用原样，无需用户配置
- 只有用户主动在管理界面修改才改变效果
- 支持 0-100 透明度调节
